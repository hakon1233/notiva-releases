# Notiva Desktop App Releases

Download the Notiva desktop app for your platform:

- **macOS (Apple Silicon)**: `Notiva-darwin-arm64.dmg`
- **macOS (Intel)**: `Notiva-darwin-x64.dmg`
- **Windows**: `Notiva-Setup.exe`

See the [Releases](https://github.com/hakon1233/notiva-releases/releases) page for downloads.
