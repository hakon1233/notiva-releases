# Research

Investigation notes, spike docs, options analysis, and technical exploration.

## What goes here

- Technical spikes and proof-of-concept findings
- Competitor or library comparisons
- Options analysis (evaluated X, Y, Z — chose Y because...)
- Deep dives into unfamiliar systems or APIs
- Performance investigations and benchmarks

## Format

Name files by topic: `auth-provider-comparison.md`, `caching-strategy-spike.md`.

Research that leads to a decision should result in an ADR in `docs/decisions/`.
